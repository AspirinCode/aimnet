## Requirements

 - [pytorch-1.1](https://pytorch.org/) 
 - [numpy](https://www.numpy.org/)
 - [pyyaml](https://pyyaml.org/)
 - [ase](https://wiki.fysik.dtu.dk/ase/)
 - [nglview](http://nglviewer.org/nglview/latest/) (for demos)
